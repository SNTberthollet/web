Les raccourcis clavier

  • Enregistrer (sauvegarder) : Ctrl + S
  • Rechercher un texte : Ctrl + F
  • Rafraîchir : Ctrl + F5
  • Fermer la fenêtre : Alt + F4
  • Rouvrir l'onglet fermé : Ctl + Maj + t
  • Changer de fenêtre : Alt + Tab
  • Nouvel onglet : Alt + Tab
  • Naviguer avec le clavier dans une fenêtre : Tab
  • Aller au début de la ligne : Déb (flèche vers en haut à droite)
  • Aller à la fin de la ligne : Fin
  • Mince les caractères s'éffacent, comment faire ? : Inser
